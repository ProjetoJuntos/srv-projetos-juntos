## How to setup and run this App locally?

1. Create a .env file such as example bellow:
```
###SERVER CONFIG###
PORT=3001
HOST=localhost
BASE_ROUTE=/api
```
2. Run commands:
```
npm install
npm run dev
```
## How to exit this App?
```
Press Ctrl + C twice, or
type .exit and press Enter, or
press Ctrl + D at the start of a line (Unix only)
```
